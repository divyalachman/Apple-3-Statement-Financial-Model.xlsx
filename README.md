# Apple-3-Statement-Financial-Model.xlsx
Excel-based three-statement financial model for Apple, including historical financials, assumptions, projected income statement, balance sheet, cash flow statement, and financial checks.

## Project Overview
This project is an Excel-based three-statement financial model for Apple Inc. The model connects the income statement, balance sheet, and cash flow statement to analyze historical financial performance and forecast future financial results.

The purpose of this project is to demonstrate financial modeling, accounting linkage, forecasting, and Excel-based analysis skills.

## Tools Used
- Microsoft Excel
- Financial Modeling
- Three-Statement Modeling
- Forecasting
- Financial Statement Analysis
- Ratio Analysis
- Balance Sheet Checks

## File Included
- `Apple-3-Statement-Financial-Model.xlsx`: Excel workbook containing the full three-statement financial model.

## Model Structure
The workbook includes multiple sections for organizing and analyzing Apple’s financial data:

### Cover Sheet
Provides the project title and model overview.

### Historical Financials
Includes historical financial statement data used as the foundation for the model.

### Assumptions
Contains forecast assumptions such as revenue growth, margins, operating expenses, tax rate, depreciation and amortization, capital expenditures, and working capital assumptions.

### Income Statement
Projects revenue, gross profit, operating expenses, operating income, taxes, net income, and earnings per share.

### Balance Sheet
Forecasts assets, liabilities, and shareholders’ equity while linking key accounts to operating assumptions and cash flow statement activity.

### Cash Flow Statement
Calculates cash flow from operations, investing activities, and financing activities, including net income, depreciation and amortization, working capital changes, capital expenditures, and financing-related items.

### Checks
Includes model checks to verify whether the balance sheet balances and whether key statement links are working correctly.

### Summary
Summarizes key financial outputs and performance indicators from the model.
